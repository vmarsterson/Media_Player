# Media_Player

Media player designed to create, play, save and show a visual representations of users' songs. 

Uses technologies including:

+ Javascript
+ HTML 
+ CSS
+ Websockets
+ Webworker
+ Geolocation services
+ apprun
+ Canvas

Used to prep for passed MTA exam

# Getting started

1. Clone and pull repository onto your computer
2. Start simple server with `http-server`
3. Find website on `localhost: 8080`
